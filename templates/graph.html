Graph Interaction

Explore your personal knowledge graph:

  • Click on a node to select it
  • Ctrl/Cmd + Click to select multiple nodes
  • Drag nodes to rearrange the graph
  • Use mouse wheel to zoom in/out
  • Click and drag the background to pan
  • Use the search bar to find specific nodes
  • Adjust node size and link strength using the sliders
  • Select two or more nodes to highlight connections and generate "what-if" scenarios
